# Bigdata infrasctucture
Hadoop, Spark, Jupyter Lab in Docker containers


Build and run all Docker containers
```sh
sh master-build.sh
```

To upload data to HDFS put content to /hdfs_upload/upload folder and run
```sh
sh hdfs_upload.sh
```
