# Bigdata infrasctucture
Hadoop, Spark, Jupyter Lab in Docker containers


Build and run all Docker containers
```sh
sh master-build.sh
```

Run MapReduce tasks
```sh
sh hw01-hdfs/job_submit.sh
```
